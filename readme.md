# Products Platform API

## Installation

### Requirements

- nodejs installed
- npm installed
- Docker installed

### Steps

#### Project Setup

- `npm i -g @nestjs/cli`
- `nest new products-platform-api`

#### Database setup
base on the `docker-compose.yml` file
- run `docker compose up -d`

#### Run The App

- `npm install`
- `npm run start:dev`
